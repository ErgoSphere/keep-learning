####居中显示DIV

1. BFC: 左右两边固定宽度，中间不设宽（自适应，随浏览器大小变化）

```html
 <div class="container">
    <div class="column"></div>
    <div class="column"></div>
    <div class="column"></div>
</div>
```
```scss
.column:nth-of-type(1), .column:nth-of-type(2) {
  width: 100px;
  height: 300px;
}
.column:nth-of-type(1) {
  float: left
}
.column:nth-of-type(2) {
  float: right
}
.column:nth-of-type(3) {
  overflow: hidden; //create BFC
  height: 300px;
}
```
